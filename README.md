# 📟 Contatore

Una semplice applicazione web che permette di incrementare, decrementare e resettare un contatore partendo dal valore iniziale di **0**.  
Il progetto è stato realizzato utilizzando **HTML**, **CSS** e **JavaScript**.

---

## 🚀 Funzionalità

- **Incremento (+)**: aumenta il valore del contatore di 1  
- **Decremento (-)**: diminuisce il valore del contatore di 1  
- **Reset**: riporta il contatore a 0  
- **Valore iniziale**: il contatore parte sempre da 0 al caricamento della pagina  

---

## 🛠️ Tecnologie utilizzate

| Linguaggio | Utilizzo |
|-----------|----------|
| **HTML** | Struttura della pagina e degli elementi del contatore |
| **CSS** | Stile, layout e presentazione grafica |
| **JavaScript** | Logica del contatore e gestione degli eventi dei pulsanti |

---

## 📂 Struttura del progetto

```
/contatore
│── index.html
│
├── /assets
│     └── favicon.ico
│
├── /css
│     └── style.css
│
└── /js
      └── script.js
```

---

## 📘 Descrizione dei file

### **index.html**
Contiene la struttura dell’interfaccia: il valore del contatore, i pulsanti (+, -, reset) e il collegamento alla favicon, al foglio di stile e allo script JavaScript.

### **/assets/favicon.ico**
L’icona dell’applicazione, mostrata nella scheda del browser.

### **/css/style.css**
Gestisce l’aspetto grafico dell’applicazione: layout, colori, spaziature e stile dei pulsanti.

### **/js/script.js**
Implementa la logica del contatore e collega i pulsanti alle rispettive funzioni tramite JavaScript.

---

## ▶️ Come utilizzare l’applicazione

1. Apri il file `index.html` in un browser  
2. Usa i pulsanti:
   - **+** per incrementare  
   - **-** per decrementare  
   - **Reset** per riportare il valore a zero  
3. Il valore del contatore si aggiorna in tempo reale
